# 🍃 KeenKeeper — Keep Your Friendships Alive

**KeenKeeper** is a professional friendship management dashboard designed to help users nurture and track their most meaningful connections. Built with a focus on minimalist design and insightful analytics, it provides a dedicated space to log interactions and visualize social habits.

---

## 🚀 Technologies Used

| Technology | Purpose |
| :--- | :--- |
| **Next.js 15** | Framework (App Router) |
| **Tailwind CSS** | Styling & Layout |
| **Recharts** | Data Visualization & Analytics |
| **React Icons** | UI Icons |
| **Geist Font** | Modern Typography |
| **Vercel** | Hosting & Deployment |

---

## ✨ Key Features

### 1. Intuitive Interaction Timeline
KeenKeeper features a clean, visually structured timeline to log calls, texts, and video chats. It helps you stay organized by keeping a clear history of when you last connected with your circle.

### 2. Professional Friendship Analytics
The app includes a dedicated Statistics page with modern **Donut Charts**. Users can see a visual breakdown of their interaction types (Calls vs. Texts vs. Videos), making it easy to understand social patterns.

### 3. Premium Figma-Inspired UI
Developed with a "Pixel-Perfect" approach, the interface features a deep bottle-green theme (`#244D3F`), soft shadows, and responsive layouts that ensure a seamless experience across all device sizes.
